## ðŸŽ¨ CUSTOMIZATION

### Change Colors
Edit `:root` variables in `styles.css`:
```css
--accent: #667eea;        /* Change primary color */
--bg-dark: #0a0e27;       /* Change background */
--error: #ff6b6b;         /* Change error color */
```

### Adjust Animations
Modify animation durations:
```css
--transition-fast: 150ms;   /* Quick transitions */
--transition-base: 300ms;   /* Standard transitions */
--transition-slow: 500ms;   /* Slow transitions */
```

### Customize Fonts
Update font-family in body:
```css
font-family: 'Your Font', sans-serif;
```
